# Overview

**FluxCore** is a desktop application that enables you to contribute your computer's idle GPU and CPU resources to the FluxEdge marketplace while earning rewards. It combines native Flux mining capabilities with the ability to rent out your hardware for pay-as-you-go computing tasks.

## What is FluxCore?

FluxCore is a user-friendly desktop client that connects your machine to the Flux decentralized network, allowing you to:

- **Provide GPU Resources**: Rent out your NVIDIA GPU to FluxEdge users for AI, ML, rendering, and compute-intensive tasks
- **Mine FLUX**: Run native Flux mining operations to earn block rewards
- **Earn Passive Income**: Generate revenue from underutilized hardware without managing complex infrastructure
- **Control Availability**: Set custom schedules for when your machine is available for rentals
- **Monitor Performance**: Track earnings, uptime, and resource utilization through an intuitive GUI

## How Does FluxCore Work?

When you install and configure FluxCore on your machine, it becomes part of the FluxEdge compute marketplace. Users renting GPU resources on FluxEdge can discover and utilize your hardware for their workloads. You maintain full control over:

- **Availability Windows**: Define specific hours when your machine can be rented
- **Resource Allocation**: Choose which GPUs and how much compute power to offer
- **Pricing**: Set competitive rates or use recommended marketplace pricing
- **Security**: All workloads run in isolated containers with strict access controls
