# Games

FluxCloud brings game server hosting to the decentralized web—giving players, creators, and communities the tools to run high-performance game servers across a global network of nodes.

Whether you're hosting for friends or managing a large community, FluxCloud provides the reliability, low latency, and flexibility you need to keep the action going.

***

### What We Support

You can currently deploy and manage the following game servers:

* **Minecraft** (Java Edition)
* **Minecraft Bedrock** (mobile, console, Windows 10/11)
* **Factorio**
* **Satisfactory**
* **Enshrouded**
* **Palworld**
* **Don't Starve Together**
* **Terraria**
* **Valheim**
* **Garry's Mod**
* **V Rising**

More titles are coming soon!
