# FluxDrive

In addition to compute resources, **FluxCloud** also offers decentralized storage powered by **IPFS**.\
FluxDrive lets you securely store, manage, and share your files with **Web3 infrastructure**, **global distribution**, and **unlimited bandwidth**.

***

### Access Methods

FluxDrive can be used two ways — both are available on every paid plan and use the same underlying storage:

* **Web UI** — manage your files in a browser at [cloud.runonflux.com/flux-drive](https://cloud.runonflux.com/flux-drive). Best for everyday use, manual uploads, and sharing links.
* **API** — programmatic access for dApps, automation, and integration into Web3 projects. See the [API Reference](api-reference.md) for endpoints and authentication.

***

### Getting Started

1. Sign in at [cloud.runonflux.com/flux-drive](https://cloud.runonflux.com/flux-drive) with your **FluxID** (Zelcore wallet).
2. Select **Sign Up** on the plan of your choice.
3. Choose your payment method:
   * **FLUX**
   * **Crypto.com**

For programmatic access, see the [API Reference](api-reference.md) for how to generate an API key and authenticate.

***

### Managing Your Files & Folders (Web UI)

Once subscribed, you can start using FluxDrive’s web interface at [cloud.runonflux.com/flux-drive](https://cloud.runonflux.com/flux-drive).

**Main Interface Features:**

* **+ New** button — Create a new folder, upload a file, or upload an entire folder via your file browser.
* **View Switch** — Toggle between list view and tile view.
* **Storage Meter** — View your current storage usage and capacity.
* **Search Bar** — Quickly find files or folders.

**Folder Actions:**

* Rename
* Delete
* Open by **double‑clicking**

**File Actions:**

* Open
* Rename
* Move to another folder
* Download
* Add new version
* Copy link (e.g. `https://fluxdrive.runonflux.io/ipfs/QmX2aMjbmMSnHEGbJ77FfwwG1agYfTSxXeFSNFQw6syBRJ`)
* Delete

**Upgrade Plan** — Use the **Upgrade Plan** button at any time to increase your storage.
